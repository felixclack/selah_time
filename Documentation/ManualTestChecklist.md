# Manual Test Checklist

Use a physical iPhone running iOS 17+.

## Onboarding
- [ ] Welcome screen loads.
- [ ] Screen Time authorization request completes (approved path).
- [ ] If denied, onboarding cannot proceed.
- [ ] App picker allows selecting apps and/or categories.
- [ ] Tradition, vibe, and translation can be set.
- [ ] Unlock duration can be adjusted (5-30 min).
- [ ] Finishing onboarding applies shields to selected apps.

## Shield + Prayer Flow
- [ ] Opening a shielded app shows Selah shield.
- [ ] "Pray to Unlock" deep-links into Selah.
- [ ] Mood selection displays prayer + scripture.
- [ ] "Amen" unlocks apps and shows confirmation.

## Unlock Window
- [ ] Apps stay unlocked for the configured duration.
- [ ] Apps re-lock after the duration expires (even if Selah is backgrounded).
- [ ] Device restart during unlock window still re-locks when time ends.

## Dashboard + Settings
- [ ] Dashboard shows Today count and Streak.
- [ ] Settings allows editing blocked apps and preferences.
- [ ] Prayer history lists recent events.

## Offline
- [ ] App functions fully with no network connectivity.
